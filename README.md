Azure-node-vision
========

## What this does:

- Shows a file upload form
- You upload an image file
- Script makes a call to azure image recognition service
- Data is received back from cognitive api service

Look how easy it is to use:

    import project
    # Get your stuff done
    project.do_stuff()
